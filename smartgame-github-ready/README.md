# SmartGame website

Статичний сайт для GitHub Pages.

## Публікація
1. Завантажте всі файли з цього архіву в корінь репозиторію `smartgame`.
2. GitHub: **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: **main**, folder: **/(root)**.
5. Збережіть і дочекайтеся публікації.

Файл `CNAME` уже містить домен `smartgame.club`.
